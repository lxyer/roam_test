- Mysql [[coalesce]]()函数 主要用来返回第一个非空参数;
    - select coalesce(a,b,c);
    - 如果a==null,则选择b；如果b==null,则选择c；如果a!=null,则选择a；如果a b c 都为null ，则返回为null（没意义）。
